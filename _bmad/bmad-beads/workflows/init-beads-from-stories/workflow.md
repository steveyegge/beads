# Init Beads from Stories Workflow\n\nBulk migrate existing BMAD story MD files to beads issues.\n\n## Purpose\n\nConvert legacy MD checklists to beads graph for mandatory beads mode.\n\n## Prerequisites\n\n- Beads initialized (`bd init`)\n- beads-sync agent loaded\n\n## Steps\n\n1. **Scan Stories**: Find all docs/stories/_.md or {output_folder}/stories/_.md\n2. **Parse & Create**: For each:\n - Extract epic/story title/ID\n - Parse checklists [ ] Task -> bd create \"Task\" --parent epic-X --priority P1\n - Sequential deps: bd dep add task2 blocks task1\n - Desc: \"Migrated from MD: [link MD]\"\n3. **Inject IDs**: Update MD metadata with beads IDs\n4. **Validate**: bd ready -- confirm hierarchy\n5. **Sync**: bd sync\n\n## Commands Template\n\n`bash\nload agent bmad-beads/beads-sync\n> full-sync\nbd list\n`\n\n## Output\n\n- beads populated with story tasks\n- MD updated with beads refs\n\nSee `_bmad/bmm/data/beads-conventions.md` for label and hierarchy conventions.
